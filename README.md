# spotify2025Wrapped
My top songs of 2025 from Spotify
This project uses unsupervised machine learning to uncover the hidden sonic blueprint of my unique musical taste by trading superficial genre tags for raw audio data. By analyzing high-dimensional audio telemetry, the pipeline groups my music into structural neighborhoods and maps out my true personal vibes. The resulting data profile visually reveals my core listening baseline, exposing my preference for high-intensity, emotionally dynamic soundscapes while filtering out the noise of traditional industry categories.
